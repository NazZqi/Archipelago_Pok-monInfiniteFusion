=========================================================
  Pokémon Infinite Fusion - Archipelago Client Mod
=========================================================

Este paquete contiene el cliente necesario para conectar 
tu juego de Pokémon Infinite Fusion con Archipelago.

INSTRUCCIONES DE INSTALACIÓN:
-----------------------------

1. Asegúrate de tener instalado Pokémon Infinite Fusion 
   (versión compatible probada: 6.7.2).

2. Descomprime todo el contenido de este archivo en la 
   carpeta RAÍZ de tu juego (la misma carpeta donde se 
   encuentra el archivo Game.exe).

3. Haz doble clic en el archivo "Instalar-Archipelago.bat".

4. El script copiará automáticamente los archivos 
   necesarios a la carpeta Data/Scripts/998_Experimental.

5. ¡Listo! Abre el juego y verás un indicador visual 
   (Círculo Verde/Rojo) en la esquina superior derecha 
   que indica el estado de conexión con Archipelago.


¿CÓMO CONECTARSE A UNA PARTIDA?
-------------------------------
1. Genera tu mundo en Archipelago usando el apworld de 
   Pokémon Infinite Fusion y tu archivo .yaml.
2. Inicia el servidor de Archipelago (ArchipelagoServer.exe).
3. Abre el cliente de Archipelago (con soporte para este 
   juego) y conéctate a la dirección del servidor.
4. Al entrar a Pokémon Infinite Fusion, el círculo 
   cambiará de rojo a verde.

¡Diviértete!
